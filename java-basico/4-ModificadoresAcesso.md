#  Modificadores de Acesso

Controlam quem pode acessar classes, atributos e métodos.
- public: acessível de qualquer lugar.
- private: acessível somente dentro da própria classe
- protected: acessível no mesmo pacote e por subclasses
- default (sem uso de palavra-chave): acessível somente dentro do pacote

## 3. Static e Final
São modificadores da linguagem Java, mudando o comportamento de classes, métodos e variáveis.
- static -> muda a quem aquilo pertence
- final -> muda o quanto aquilo pode mudar
